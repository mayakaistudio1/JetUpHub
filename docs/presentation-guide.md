# Гайд по работе с презентацией — для новых участников

Привет! Этот документ поможет тебе разобраться, где что лежит и как работать с интерактивной презентацией в проекте.

---

## Как начать работу в Replit

1. Открой проект по ссылке, которую тебе прислали
2. Слева — панель с файлами (как папки на компьютере)
3. Нажми на файл — он откроется в редакторе справа
4. Все изменения сохраняются автоматически
5. Приложение перезагружается само после каждого изменения — результат видно сразу в превью справа

---

## Как посмотреть презентацию

1. В превью (окно справа) перейди по адресу `/dennis`
2. Нажми кнопку **«Чат с Dennis AI»**
3. В чате нажми кнопку **«Начать презентацию»** (или аналогичную кнопку с искрой)
4. Откроется интерактивная презентация — листай слайды влево/вправо

## Как устроена презентация (кратко)

- **10 слайдов** — каждый описан в функции `buildSlides()` в файле `PresentationOverlay.tsx`
- Каждый слайд имеет: заголовок, текст, картинку, цвет акцента, чипы (кнопки для вопросов AI)
- Некоторые слайды имеют **интерактивные элементы** — карточки, которые можно нажать для подробностей
- Все тексты берутся через функцию `t('pdh.s1.title')` — это ключ перевода из `LanguageContext.tsx`

## Как менять текст слайда

Тексты находятся в файле `client/src/contexts/LanguageContext.tsx`.

Например, чтобы изменить заголовок первого слайда, найди в файле:

```
'pdh.s1.title': 'Текст заголовка...',
```

Он будет в трёх блоках — для каждого языка (ru, de, en). Не забудь поменять во всех трёх.

---

## Как менять стили (внешний вид)

Стили презентации находятся в `client/src/pdh-v3.css`.

Классы начинаются с `pres-` (например, `pres-card`, `pres-title`, `pres-journey-bar`).

Если хочешь поменять цвет, размер шрифта, отступы — ищи нужный класс в этом файле.

---

## Важные правила

1. **Работай только с файлами презентации** (перечислены выше). Не трогай другие файлы проекта — они отвечают за основной сайт
2. **Не удаляй файлы** — только редактируй содержимое
3. **Если что-то сломалось** — не паникуй. Replit сохраняет историю изменений, и всё можно откатить
4. **Проверяй результат** в превью после каждого изменения

## Структура слайда в коде (для понимания)

```typescript
{
  id: 1,                              // номер слайда
  title: t('pdh.s1.title'),           // заголовок (из переводов)
  text: t('pdh.s1.text'),             // основной текст
  image: '/images/presentation/scene_01.png',  // картинка
  accent: '#7C3AED',                  // цвет акцента
  type: 'standard',                   // тип слайда
  chips: [                            // кнопки-вопросы внизу
    { text: t('pdh.s1.c1'), intent: 'REALITY_WHY' },
    { text: t('pdh.s1.c2'), intent: 'REALITY_MISTAKES' },
  ],
}
```
