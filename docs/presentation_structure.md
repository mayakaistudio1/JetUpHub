## Технические детали

### Анимированный фон (FinancialBackground.tsx)
* 3 слоя: CSS радиальный градиент + Canvas сетевые линии + Canvas рыночные свечи
* Параллакс: слои смещаются по translateY при переключении слайдов (~10px)
* Цикл анимации: 3-6 секунд, очень медленное движение
* Без видео — лёгкий, оптимизирован для Telegram MiniApp

### Интерактивная карта (EcosystemMapSlide.tsx)
* SVG линии + Framer Motion узлы
* Каждый узел кликабельный → glassmorphism info card
* Кнопка "Спросить Dennis" → отправляет intent в AI-чат

### Glassmorphism-карточка
* `background: rgba(8, 6, 20, 0.60)` (лёгкая, прозрачная)
* `backdrop-filter: blur(20px)`
* `border-radius: 24px`
* `border: 1px solid rgba(255, 255, 255, 0.10)`

### Чипы
* 2-3 чипа на слайд, каждый с `{ text, intent }`
* Нажатие → карточка пульсирует (scale 1→1.02) + открывается чат с контекстом слайда
* AI-ответ с контекстом текущего слайда (system message injection)

### Интерактивные элементы слайдов
* `interactiveType` + `interactiveItems[]` на слайдах 4, 5, 6, 8
* 4 типа: `security-points` (pill badges), `strategy-cards` (3 горизонтальные карточки), `graph-points` (SVG график + пульсирующие точки), `ai-nodes` (сетка 2x2 узлов)
* Нажатие на элемент → `MicroInfoCard` (fixed, centered, glassmorphism) с описанием
* CSS классы: `.si-*` в `pdh-v3.css`

### Анимации
* Карточка: opacity 0→1, translateY 18→0, scale 0.98→1, duration 450ms
* Параллакс фона: translateY сдвиг при переключении слайдов
* Узлы экосистемы: пульсация scale 1→1.05, 3-4 сек цикл

### Визуальный стиль сцен
* Cinematic fintech documentary (Apple keynote / Bloomberg / Netflix style)
* Реалистичный бизнесмен/силуэт внутри финансовых данных
* Без particle silhouettes, AI роботов, цифровых гуманоидов
